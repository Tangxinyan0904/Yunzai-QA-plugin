# 🎭 Yunzai-QA-plugin

一款为 **Yunzai-Bot** 打造的轻量级、响应迅速的问答管理插件。支持文本与图片的灵活配置，采用 Base64 编码确保图片下发的高成功率。

---

## ✨ 核心特性

- 🚀 **极速响应**：基于内存缓存优化，问答匹配快如闪电。
- 🖼️ **图文支持**：深度优化图片下发逻辑，支持 Base64 编码分发。
- 🛠️ **简单易用**：纯净的指令设计，无需复杂配置，上手即玩。
- 📂 **结构清晰**：数据隔离存储，方便随时备份与迁移。

---

## 🛠️ 安装说明

请务必按照以下路径进行安装，以确保插件正常运行：

1. **放置脚本**：
   将 `QA.js` 放置在云仔根目录下的：
   `plugins/example/` 文件夹中。

2. **下载数据文件夹（关键！）**：
   请务必下载并放置 **`QADATA`** 文件夹。该文件夹需同样放置在：
   `plugins/example/` 目录下。

> [!CAUTION]
> **注意**：如果缺少 `QADATA` 文件夹，插件在保存图片或持久化数据时会报错哦！

## 🎮 指令说明

| 指令 | 格式 | 作用 | 备注 |
| **设置问答** | `添加模糊问问题1答回答1 | 创建或更新一条问答 | 答案处可发送图片 |
| **删除问答** | `#删除问答 问题` | 彻底移除指定的问题 | —— |
| **问答列表** | `#问答列表` | 查看当前所有已记录的问答 | —— |
| **开启问答** | `#开启问答` | 在当前群聊启用问答响应 | 仅限管理员/主人 |
| **关闭问答** | `#关闭问答` | 暂时停止当前群聊的问答响应 | 仅限管理员/主人 |
