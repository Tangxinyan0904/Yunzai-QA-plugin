# 🎭 Yunzai-QA-plugin (QA全能问答)

一款为 **Yunzai-Bot** 打造的轻量级问答管理插件。支持**模糊/精确**双匹配模式，内置 CD 保护，并深度优化了图片存储与 Base64 下发逻辑。

---

## ✨ 核心特性

- 🧠 **双匹配模式**：支持 `模糊问`（包含关键词即触发）和 `精确问`（必须完全一致才触发）。
- 🖼️ **图文记忆**：支持在添加问答时直接发送图片，插件会自动持久化存储。
- 🛡️ **内置 CD**：默认 3 秒冷却，防止恶意刷屏（可在代码魔法配置区修改）。
- 📦 **跨环境兼容**：发送图片时自动转为 Base64 编码，完美解决不同操作系统路径不通的问题。

---

## 🛠️ 安装说明

请务必按照以下路径进行安装：

1. **放置脚本**：
   将 `QA.js` 放置在云仔根目录下的：
   `plugins/example/` 文件夹中。

2. **下载数据文件夹（必看！）**：
   请确保在 `plugins/example/` 目录下存在 **`qadata`** 文件夹。

## 🎮 指令指南

| 指令格式 | 作用说明 | 示例 |
| :--- | :--- | :--- |
| **`~添加模糊问[问题]答[答案]`** | 只要消息中**包含**问题词，即触发回答 | `~添加模糊问你好答嘿嘿，棉宝在哒！` |
| **`~添加精确问[问题]答[答案]`** | 消息必须与问题词**完全一致**才触发 | `~添加精确问笨蛋答你才是笨蛋！(｡>ㅅ<｡)` |
| **`~问答列表`** | 分页查看所有已记录的问答编号与内容 | `~问答列表` |
| **`~删除问答[编号]`** | 根据列表编号删除（支持批量，用逗号分隔） | `~删除问答1,3,5` |

> [!TIP]
> **小技巧**：在执行 `~添加...答` 指令时，如果消息里带有图片，也会自动把图片也记下来哦！

## ⚙️ 群聊配置

您可以直接打开 `QA.js` 顶部的“魔法配置区”进行个性化调整：
- `CD_SECONDS`: 调整触发冷却时间（默认 3 秒）。
- `ALLOWED_GROUPS`:填入群号开启白名单模式，留空 `[]` 则全群生效。多个参数使用，分割
- `ENABLE_PRIVATE`: 是否开启私聊响应
